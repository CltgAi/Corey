# Corey New AI – Web Dashboard

This is the full project including:

- Dashboard  
- Finance (Stocks, Bonds, ETFs)  
- Crypto (Top 50 live prices)  
- CSV Upload  

## Installation

1. Upload entire folder to GitHub repo root.
2. Go to Settings → Pages → Select Branch = main.
3. Wait 30 sec and visit:
   https://USERNAME.github.io/corey_new_ai/
